## Build 
Clone the project

```
git clone git@github.com:rickymaciel/login-sass.git
cd login-sass
```


Install NPM

```
npm install
```

## Compile sass

Compressed
```
npm run scss
```

Nested
```
npm run scss-n
```

Compact
```
npm run scss-p
```

Expanded
```
npm run scss-e
```
